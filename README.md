# e-com website
 This is simple e-commerce website.

## Requirements
- Node >= 12
- npm >= 8

## Project Setup
 - Git Clone: https://github.com/chamikabrainstackhub/eCom-Website.git
 - After that, `npm install` 
 - To run the project `npm start`

## Modules
- Home page
- About us Page
- Contact Page
- Product Details Page
- Product List Page
- Cart
- Checkout Page
- Profile
